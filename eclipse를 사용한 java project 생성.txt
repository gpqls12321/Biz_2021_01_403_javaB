eclipse를 사용한 java project 생성
1. project 만들기
   가. new / java Project
   - project는 어플리케이션 이름
   - 영문, 숫자 등을 조합하여 만듦

2. package 생성
   가. project 생성 후 최초 package 생성
   나. packaga 이름은 회사의 Domain Name 을 거꾸로 시작하여 작성
       예) https://callor.com 이면 com.callor.hello
           -3단계 package 라고 한다.
   다. 3단계 package 를 base package 라고 부른다.







문법오류 : syntax error
  - java 문법 규칙을 지키면서 코딩을 할 때 대소문자, 빈칸, 세미콜론(;), 괄호(열고, 닫기의 쌍) 등이 
    규칙에 맞지 않을 때 eclipse는 자동으로 감지를 하고 알려준다. 
  - 문법오류가 발생하면 명령어, 명령문에 빨간 밑줄이 나타난다.
